# Drinkys

Un buscador de bebidas, utiliza react-router-dom para multiples paginas, zod para validaciones, zustand para estados globales, tailwind para estilos, axios para fetch de API.

Crear multiples slides pero consumirlos desde un solo store

Agregar una forma de cerrar notificaciones utilizando setTimeOut desde notificationsSlice

sessionStorage y load inicial

Se utiliza lazy y suspense para mejorar la carga de paginas, esto para tener mejor rendimiento de la aplicación, se nota más cuando se tienen muchas paginas, pero este es un buen ejemplo de como hacerlo.
